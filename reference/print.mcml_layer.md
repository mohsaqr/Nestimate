# Print Method for an mcml Layer

Compact summary of one mcml layer (macro or a single within-cluster
network) – nodes, edges, weight matrix dimensions – without spilling the
full `$weights`, `$inits`, or `$data` contents.

## Usage

``` r
# S3 method for class 'mcml_layer'
print(x, ...)
```

## Arguments

- x:

  An `mcml_layer`.

- ...:

  Unused.

## Value

The input, invisibly.
